DOWNLOAD

BookiePro.fun is available for download on Windows, Mac and Linux platforms.

Download the BookiePro.fun software and click ‘Save File’ when prompted. The software will be saved to either your desktop or ‘Downloads’ folder.

Windows

Go to your Windows desktop or the location where you chose to save the program. Double click on the BookiePro.fun icon.

Allow the application to install. When it is complete, the app will launch.

Some anti-virus programs may require you to approve the app before launching. In this case, clicking “Advanced Settings” followed by “Run Anyway” will allow the app to run.

Mac

Go to your Downloads folder or the location where you chose to save the BookiePro.fun software. Double click to run.

Allow the application to install. When it’s complete, drag the BookiePro.fun icon into the Applications folder.

Navigate to the Applications folder and launch the BookiePro.fun app.

If you receive this error:

Go to Apple Menu > System Preferences. Click on Security & Privacy, then select the General tab and click “Open Anyway”.

BookiePro.fun will now launch.

Linux

Go to your Linux desktop

And make note of where the file downloaded to.

Open up your Console

and run the following command

$ sudo dpkg -i ~//BookiePro.fun_0.29.0-F_amd64.deb

Make sure you replace // with the correct file path.

BookiePro.fun will now launch.
